# <a name="client-features-in-exchange-online-archiving"></a><span data-ttu-id="ce019-106">Funzionalità client in Archiviazione Exchange Online</span><span class="sxs-lookup"><span data-stu-id="ce019-106">Client Features in Exchange Online Archiving</span></span>

All network connectivity to the user's archive occurs over the Internet, and virtual private network (VPN) connections are not required. Organizations can publish an on-premises Client Access server to allow users to access their primary mailbox using Outlook Anywhere, without requiring a VPN connection. If VPN access is required to access the user's primary mailbox located on an on-premises server, this requirement does not change.</span></span>
