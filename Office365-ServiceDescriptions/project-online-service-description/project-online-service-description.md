# <a name="project-online-service-description"></a>Descrizione del servizio Project Online

Project Online è una flessibile soluzione online per la gestione dei portfolio di progetti (PPM) e il lavoro quotidiano. Recapitato a Office 365, Project Online offre funzionalità di gestione progetto potenti per la pianificazione, definizione della priorità e la gestione di progetti e gli investimenti portfolio di progetti, ovvero praticamente ovunque quasi qualsiasi dispositivo. Project Online consente agli amministratori, Project Manager e visualizzatori, i manager di progetto e delle risorse e responsabili del team e membri. 
  
Sono disponibili tre piani per Project Online: Project Online Essentials, Project Online Professional e Project Online Premium.
  
Per confrontare le funzionalità dei vari piani, vedere [Confronto tra i piani di Project Online](https://products.office.com/en-us/project/compare-microsoft-project-management-software).
  
È possibile esportare, salvare e stampare le pagine della descrizione del servizio Office 365. Ulteriori informazioni su come [esportare più pagine](https://go.microsoft.com/fwlink/?LinkId=403349).
  
## <a name="project-online-boundaries-and-limits"></a>Limiti di Project Online

Il servizio Project Online prevede alcune limitazioni. Per i dettagli, vedere [Project Online: limiti del software](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489).
  
## <a name="data-backup"></a>Backup dei dati

 Project Online si basa su SharePoint Online e segue lo stesso criterio sul backup dei dati previsto per SharePoint. Per maggiori informazioni, consultare la sezione Backup di dati nella [Descrizione del servizio SharePoint Online](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx).
  
## <a name="use-of-sharepoint-online"></a>Utilizzo di SharePoint Online

Il servizio di Project Online richiede l'utilizzo di SharePoint Online, verrà eseguito il provisioning automaticamente come parte del servizio Project Online. Diritti per la funzionalità di SharePoint Online con Project Online Professional o Premium disponibili sono un limite di archiviazione e accesso ai dati per il supporto di Project Online.
  
## <a name="licensing-considerations"></a>Considerazioni sulle licenze

### <a name="internal-users"></a>Utenti interni

Alcuna interazione in un sito di Project Online richiede una licenza Project Online.
  
- Project Online Essentials è il requisito minimo.
    
- Per più avanzati casi di utilizzo, gli utenti devono disporre di licenze appropriate. Project Manager è necessaria almeno una licenza di Project Professional in linea. Program Manager è necessaria una licenza Project Online Premium.
    
> [!IMPORTANT]
>  Project Online Essentials è una licenza di *componente aggiuntivo* di membri del team per i clienti che dispongono di Project Online Professional o Project Online Premium.<br/>Scadenza della licenza di Project Online Professional o Project Online Premium ultima, le istanze di Project Web App verranno eliminate dopo 120 giorni.<br/>Per sottoscrizioni di valutazione di Project Online, le istanze di Project Web App verranno eliminate dopo il periodo di valutazione di 30 giorni. 
  
### <a name="external-users"></a>Utenti esterni

Project Online licenze per gli utenti esterni segue lo stesso criterio a quella per gli utenti interni. Alcuna interazione in un sito di Project Online richiede una licenza Project Online.
